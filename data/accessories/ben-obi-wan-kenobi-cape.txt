Accessory Name: Ben (Obi-Wan) Kenobi Cape

Category:
Soft Goods – Vinyl Cape

Used With:
- Ben (Obi-Wan) Kenobi figure (Kenner)
- Regional variants (Lili Ledy, PBP, etc.)

--------------------------------------------------

OVERVIEW

The Ben Kenobi cape is a brown vinyl soft goods accessory included with the Obi-Wan Kenobi figure.

It exists in multiple clearly defined mould variants (M1 onwards), with strong and visible differences between them.

Compared to Leia, Ben capes show more obvious variation in:

- Hole layout and spacing
- Top curve shape
- Overall proportions
- Vinyl cut and finish

--------------------------------------------------

CRITICAL COLLECTOR NOTE

👉 Ben cape variants are EASY to spot once you know what to look for  
👉 Hole layout is the primary identifier  
👉 Shape differences are more obvious than Leia  

--------------------------------------------------

MATERIAL

- Brown vinyl
- Flexible but varies in thickness
- Can stiffen, crease, or curl with age

--------------------------------------------------

COLOUR

Typical:

- Medium brown
- Reddish brown
- Dark brown

IMPORTANT:
👉 Colour variation is often batch or ageing related, not always mould-specific

--------------------------------------------------

CORE IDENTIFICATION FEATURES

Focus on:

1. Arm hole spacing and alignment
2. Arm hole size differences
3. Top curve shape
4. Width and height proportions
5. Edge cut quality
6. Vinyl thickness

--------------------------------------------------

ARM HOLE CONFIGURATION (CRITICAL)

Main identifier across variants:

- Two arm holes (standard across all vinyl capes)
- Spacing between holes
- Alignment (straight vs offset)
- Hole size and shape
- Clean vs rough punch edges

👉 This is one of the primary ways to identify mould variants

MOULD VARIANTS (M FAMILIES)

The Ben cape is classified into multiple mould families.

-----------------------------------

M1

- Early production
- Typically:
  - Clean hole punches
  - Balanced spacing
  - Symmetrical top curve
- Often associated with early Kenner releases

-----------------------------------

M2

- Variation of M1
- Differences in:
  - Hole placement
  - Slight shape change
- Subtle but noticeable side-by-side

-----------------------------------

M3 (LILI LEDY)

- Associated with Lili Ledy production
- Distinct:
  - Hole layout variation
  - Slightly different vinyl tone
- Often less consistent finishing

-----------------------------------

M4 (UNITOY / OTHER)

- Further variation in:
  - Hole spacing
  - Curve shape
- Slightly different proportions

-----------------------------------

M5+

- Increasing variation across moulds
- Includes:
  - Wider capes
  - Different curve profiles
  - Irregular hole spacing

-----------------------------------

LATER / REGIONAL VARIANTS

- Greater inconsistency
- Often:
  - Rougher cut edges
  - Less precise hole punching
  - More variation in vinyl tone

--------------------------------------------------

HOLE CONFIGURATION (CRITICAL)

Main identifier across variants:

- Number of holes (usually 2, but spacing varies)
- Distance between holes
- Alignment (straight vs slightly offset)
- Size differences between holes

👉 This is the fastest way to identify mould family

--------------------------------------------------

SHAPE VARIATION

Key differences:

- Top curve:
  - More rounded (early)
  - Flatter or wider (later)

- Overall silhouette:
  - Narrower vs wider
  - Taller vs shorter drop

--------------------------------------------------

EDGE FINISH

- Clean cut (early moulds)
- Slight roughness (later)
- Occasional uneven trimming

--------------------------------------------------

SURFACE

- Generally smooth
- Some variants show:
  - Slight texture
  - Minor surface imperfections

--------------------------------------------------

FACTORY ASSOCIATIONS

General tendencies:

- Kenner → early M1 / M2
- Lili Ledy → M3
- Unitoy → mid variants
- PBP / other → later regional variation

(Not absolute – always verify physically)

--------------------------------------------------

IDENTIFICATION STRATEGY

1. Count and measure hole spacing
2. Compare hole alignment
3. Check top curve shape
4. Assess overall width and drop
5. Compare edge cut quality
6. Cross-reference with M variant groups

--------------------------------------------------

COMMON MISTAKES

- Treating Ben capes like Leia capes (they are not as subtle)
- Ignoring hole placement differences
- Using colour as the main identifier
- Not matching cape to COO figure

--------------------------------------------------

REPRODUCTION WARNING

Repro Ben capes often show:

- Incorrect hole spacing
- Wrong proportions
- Vinyl too thick or too stiff
- Poor cut edges
- Incorrect colour tone

--------------------------------------------------

COLLECTOR INSIGHT

- Ben cape is one of the easier capes to classify correctly
- Hole layout makes identification straightforward
- Correct pairing improves figure accuracy and value
